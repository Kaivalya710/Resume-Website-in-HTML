# Resume Website

## Description

A clean, semantic HTML-only resume website showcasing professional experience, skills, projects, and education. This single-page site demonstrates the power of HTML semantics without any CSS styling, maintaining excellent readability and structure.

## Features

- Semantic HTML5 markup
- Responsive structure using only HTML elements
- Clear sectioning with proper heading hierarchy
- Tabular data for experience and education
- Professional contact information
- Copyright footer

## Technologies Used

- HTML5 (no CSS, no JavaScript)
- Semantic elements: `<header>`, `<section>`, `<footer>`, `<table>`, `<ul>`, `<li>`
- Horizontal rules (`<hr>`) for visual separation

## Setup Instructions

### Local Development

1. **Clone the repository**
    
    ```bash
	    git clone https://github.com/Kaivalya710/Resume-Website-in-HTML.git
	    cd Resume-Website-in-HTML
    ```
    
2. **Open the file**
    
    - Simply open `index.html` in any web browser
	
3. **View in browser**
    
    - Install Live Server on VS Code
    - Click `index.html` to open and click on "Open with Live Server"

## Usage

This resume website is designed to be:

- Easy to customize with your own information
- Simple to maintain (pure HTML)

### Customization

Edit the `index.html` file to update:

- Personal information (name, title, contact details)
- About Me section
- Skills list
- Work experience entries
- Project descriptions
- Education details

## Deployment on GitHub Pages

### Step-by-Step Guide

1. **Create a GitHub repository**
    
    ```bash
    # Initialize git
    git init
    
    # Add your files
	    git add index.html
    
    # Commit
    git commit -m "single-page resume website in HTML"
    ```
    
2. **Push to GitHub**
    
    ```bash
    # Add remote repository
    git remote add origin https://github.com/yourusername/Resume-Website-in-HTML.git
    
    # Push to main branch
    git branch -M main
    git push -u origin main
    ```
    
3. **Enable GitHub Pages**
    
    - Go to your repository on GitHub
    - Click on **Settings** tab
    - Scroll down to **Pages** section (in the left sidebar)
    - Under **Source**, select **main** branch
    - Select **/ (root)** folder
    - Click **Save**

5. **Access your site**
    
    - Your site will be live at: `https://yourusername.github.io/resume-website/`
    - It may take a few minutes for the site to deploy
